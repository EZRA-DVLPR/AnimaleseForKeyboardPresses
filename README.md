# AnimaleseForKeyboardPresses
Makes the respective sound from animalese whenever the user presses the corresponding letter on their keyboard.

# How To Use

## Windows Instructions

1. Download the compressed folder under releases
2. Unzip the folder
3. Open the `animalesetyper` folder
4. Locate the `animalesetyper.exe` file
5. Create a shortcut
6. Move the shortcut to desired location
7. Open and execute shortcut
8. There will be a new empty terminal window that opens. Leave this running in the background while you do other tasks.
9. Enjoy typing with animalese
